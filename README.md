# Amazon clone projects with HTML and CSS.

a complete front-end replica of Amazon.com using HTML5 and CSS3, recreating key interface sections such as the header, search bar, product listings, and footer to demonstrate practical UI development skills.

Translated a complex real-world web design into clean, structured code by leveraging CSS Flexbox, positioning, and the Box Model — building all layouts manually without relying on frameworks like Bootstrap or Tailwind.
